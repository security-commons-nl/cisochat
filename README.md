# cisochat

> Een AI-assistent die redeneert vanuit het perspectief van een CISO in de publieke sector.

Een CISO bij een gemeente heeft dagelijks vragen die niet met een Google-zoekopdracht beantwoord worden. cisochat geeft antwoorden vanuit de gedachtegang van een ervaren CISO in de Nederlandse publieke sector — gegrond in BIO 2.0, ISO 27001, ISO 27701, ISO 22301 en AVG — lokaal gehost, zonder data buiten de EU.

## Principes

Dit project volgt de [architectuur- en communityprincipes](https://github.com/security-commons-nl/.github/blob/main/PRINCIPLES.md) van security-commons-nl: EU-soevereiniteit, AI altijd adviserend, auditbaarheid by design, least privilege en open source als standaard.
